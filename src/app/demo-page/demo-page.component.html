Content Visibility Demo

What is content-visibility?

The CSS content-visibility property enables us to skip rendering of off-screen content until it's needed. This can significantly improve page load performance for long scrolling pages.

How to use it:

.my-component {{ '{' }}
  content-visibility: auto;
  contain-intrinsic-size: 400px;
{{ '}' }}

The contain-intrinsic-size property helps prevent layout shifts by telling the browser the approximate size of the hidden content.
